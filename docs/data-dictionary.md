# 数据字典

> A股AI投研系统 — 所有字段含义、判断标准、数据来源

## 基本面（来源：Efinance + Baostock，季度更新）

| 字段 | 含义 | 判断标准 | 影响 |
|---|---|---|---|
| **PE（市盈率）** | 股价 ÷ 每股收益。每赚 1 元钱需要付多少股价 | <20 低估 · 20-50 合理 · >50 高估 | 估值师 Agent |
| **PB（市净率）** | 股价 ÷ 每股净资产。市场给净资产的溢价倍数 | <1 破净 · 1-3 正常 · >5 高估 | 估值师 Agent |
| **ROE（净资产收益率）** | 净利润 ÷ 净资产。公司用股东的钱赚了多少 | >15% 优秀 · 5-15% 一般 · <5% 差 | 基本面评分 25% |
| **营收** | 公司主营业务收入（元） | 越大越好，但需结合市值 | — |
| **营收增速** | 相比去年同期的增长率 | >20% 高增长 · 0-20% 稳增长 · <0 下滑 | 基本面评分 25% |
| **净利润** | 扣除所有成本后的利润（元） | — | — |
| **净利润增速** | 相比去年同期的增长率 | >20% 高增长 · < -30% 暴跌（降级） | 基本面评分 25% |
| **EPS（每股收益）** | 净利润 ÷ 总股本 | — | — |
| **资产负债率** | 总负债 ÷ 总资产 | <60% 安全 · 60-80% 注意 · >80% 高风险 | 基本面评分 10% |
| **市值** | 股价 × 总股本（元） | — | — |
| **行业** | 申万/证监会行业分类 | — | 行业集中度/对比 |

## 技术面（来源：daily_prices 计算，每日更新）

| 字段 | 含义 | 判断标准 |
|---|---|---|
| **MA20（20日均线）** | 最近 20 个交易日收盘价的平均值 | close > MA20 短期向好；< MA20 短期转弱 |
| **MA60（60日均线）** | 最近 60 个交易日收盘价的平均值 | close > MA60 中期向好；< MA60 中期转弱 |
| **趋势评分** | 基于均线排列的技术面评分（0-100） | >80 强劲多头 · 50-80 向好 · <50 弱势 |
| **动量评分** | 基于近期收益率的动量评分（0-100） | >80 动量充沛 · <30 动能衰竭 |
| **流动性评分** | 基于成交额的流动性评分（0-100） | >80 活跃 · <30 清淡 |
| **风险评分** | 基于波动率和回撤的风险控制评分（0-100） | >80 低风险 · <30 高风险 |
| **总评分** | 趋势×0.35+动量×0.25+流动×0.20+风险×0.20+基本面×0.2 | — |

## 行情（来源：AkShare sina，每日更新）

| 字段 | 含义 |
|---|---|
| **开盘价** | 当交易日第一笔成交价 |
| **最高价** | 当交易日最高成交价 |
| **最低价** | 当交易日最低成交价 |
| **收盘价** | 当交易日最后一笔成交价 |
| **成交量** | 当交易日成交股数 |
| **成交额** | 成交量 × 成交均价（元） |
| **换手率** | 成交量 ÷ 流通股本。衡量交易活跃度 | >2% 活跃 · 0.5-2% 正常 · <0.5% 清淡 |

## 观察池（来源：watchlist 表，每日更新）

| 字段 | 含义 | 判断标准 |
|---|---|---|
| **入场价** | 首次推荐时的收盘价 | — |
| **最新价** | 最近交易日收盘价 | — |
| **收益** | (最新价 - 入场价) ÷ 入场价 | 正=盈利 负=亏损 |
| **最大回撤** | 入场后最大亏损幅度 | < -10% 需关注 |
| **破趋势线** | 最新价是否跌破 MA20 或 MA60 | 是 ⚠️ 趋势转弱 |
| **跟踪天数** | 从首次推荐到现在的交易日数 | 5-20 天 |
| **状态** | 观察中/降级/退出/阻断 | — |

## 复盘（来源：复盘 API，实时计算）

| 字段 | 含义 | 判断标准 |
|---|---|---|
| **推荐后 N 日收益** | 推荐日后第 N 天的累计涨跌幅 | >0 推荐有效；<0 推荐失效 |
| **胜率** | 推荐中盈利的比例 | >60% 优秀 |
| **盈亏比** | 平均盈利 ÷ 平均亏损 | >1.5 优秀 |
| **Sharpe Ratio** | 风险调整后收益。每承担 1 单位风险获得多少超额回报 | >1 优秀 · 0.5-1 良好 · <0 差 |
| **因子 IC** | 因子值与未来收益的相关性。衡量因子是否有效 | >0 因子有效 · <0 因子与预期相反 |
| **换手率** | 本次推荐与上次推荐的重叠比例 | 低=稳定 高=轮动快 |
| **行业集中度** | 推荐中最大行业占比 | >50% 集中度过高 |

## Agent 委员会（来源：DeepSeek API，4 轮辩论）

| 字段 | 含义 | 判断标准 |
|---|---|---|
| **AI 评级** | 5 个 Agent 四轮辩论后的投票结果（1-5 分） | 5=强烈推荐 4=推荐 3=中性 2=谨慎 1=回避 |
| **共识** | Agent 投票一致程度 | 全票通过=5:0 · 多数同意=4:1 或 3:2 · 分歧=2:3 · 否决=1:4 或 0:5 |
| **辩论报告** | 完整的四轮辩论记录 | 含每个 Agent 的观点/搜索证据/投票 |

## 常见问题

**Q: 为什么 PE/PB 有时显示"—"？**
A: PE/PB 数据来自 Efinance，受网络代理限制，部分股票可能缺失。

**Q: 因子评分为什么和我直觉不符？**
A: 评分基于定量公式（如 MA20/MA60 排列、收益率），不是人工判断。如果觉得不准，可以调整 config.json 中的因子权重。

**Q: Agent 评级和规则评级有什么区别？**
A: 规则评级（A/B）基于公式计算；Agent 评级（1-5）是 5 个 AI 角色辩论后的投票结果。两者可能不一致——不一致本身就是有价值的信息。

**Q: Sharpe Ratio 什么时候才能有数据？**
A: 需要至少 2 个交易日的前向价格。系统刚搭建时会是"—"，使用一段时间后自动出现。
