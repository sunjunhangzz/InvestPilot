# 定时任务管理

项目通过 macOS launchd 实现每天 08:00 早盘推送和 12:00 午间推送。

## 安装（首次）

```bash
# 从项目根目录执行
cp scripts/com.investpilot.morning.plist ~/Library/LaunchAgents/
cp scripts/com.investpilot.noon.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.investpilot.morning.plist
launchctl load ~/Library/LaunchAgents/com.investpilot.noon.plist
```

## 查看状态

```bash
# 查看是否在运行
launchctl list | grep investpilot

# 输出示例：
# -   0   com.investpilot.morning
# -   0   com.investpilot.noon
# 第二列是退出码，0 表示上次正常，- 表示还没运行过
```

## 修改时间

```bash
# 1. 停掉旧任务
launchctl unload ~/Library/LaunchAgents/com.investpilot.morning.plist

# 2. 编辑项目里的模板（改 Hour/Minute）
nano scripts/com.investpilot.morning.plist

# 3. 重新部署
cp scripts/com.investpilot.morning.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.investpilot.morning.plist
```

> 时间使用 24 小时制。Hour=14, Minute=30 → 下午 2:30。

## 修改推送内容

```bash
# 直接改脚本，无需重启定时任务
nano app/worker/scripts/send_report.py
# 下次触发时自动用新代码
```

## 停用

```bash
launchctl unload ~/Library/LaunchAgents/com.investpilot.morning.plist
launchctl unload ~/Library/LaunchAgents/com.investpilot.noon.plist
```

## 休眠 & 唤醒

- 电脑休眠期间定时任务**不执行**
- 唤醒后**不会自动补跑**（`RunAtLoad=false`）
- 如果错过推送时间，手动执行：
  ```bash
  bash scripts/morning_push.sh
  bash scripts/noon_push.sh
  ```

## 手动测试

```bash
bash scripts/morning_push.sh   # 完整流程：分析 + 早盘推送
bash scripts/noon_push.sh      # 午间增量更新 + 涨跌幅推送
python app/worker/scripts/send_report.py --type=morning --dry-run  # 只看不推送
```

## 相关文件

| 文件 | 说明 |
|---|---|
| `scripts/com.investpilot.morning.plist` | 早盘定时模板 |
| `scripts/com.investpilot.noon.plist` | 午间定时模板 |
| `scripts/morning_push.sh` | 早盘触发脚本 |
| `scripts/noon_push.sh` | 午间触发脚本 |
| `app/worker/scripts/send_report.py` | 推送核心逻辑 |
| `~/Library/LaunchAgents/` | macOS 系统目录（安装后） |
