# 本地启动说明

## 前置环境

- Python 3.12+
- Node.js 22+
- npm 10+

## 1. 安装依赖

```bash
cd app/worker
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

cd ../web
npm install
```

## 2. 配置

```bash
cp .env.example .env
# 如需 AI 报告，编辑 .env 填入 DEEPSEEK_API_KEY
```

## 3. 初始化 + 采集 + 筛选

```bash
source app/worker/.venv/bin/activate

# 数据库
python app/worker/scripts/init_db.py
python app/worker/scripts/check_schema.py

# 数据采集
python app/worker/scripts/update_stocks.py
python app/worker/scripts/update_prices.py --codes=000001,600519  # 全量去掉 --codes

# 因子 + 推荐
python app/worker/scripts/calc_factors.py
python app/worker/scripts/run_screening.py
python app/worker/scripts/update_watchlist.py

# AI 报告（可选，需 API Key）
python app/worker/scripts/generate_report.py
```

## 4. 启动 Web

```bash
cd app/web
npm run dev
```

浏览器打开 `http://localhost:3000/dashboard`

## 验证

```bash
python app/worker/scripts/check_schema.py   # Schema validation: PASSED
cd app/web && npm run lint && npm run typecheck
```
