# 代码层开发规范索引

## 目标

本目录沉淀代码层面的开发规范，用来约束后续实现过程。

这些规范参考了：

- Python PEP 8
- Python logging 官方文档
- TypeScript Handbook
- React 官方文档
- Next.js 官方项目结构文档
- ESLint 官方文档
- Prettier 官方文档
- SQLite 官方文档
- OWASP Top 10

## 强制原则

- 后续写代码必须遵守本目录规范。
- 如果规范和实际代码冲突，先更新规范，再改代码。
- 如果规范和外部库约定冲突，优先遵守外部库官方约定，并在文档中说明。
