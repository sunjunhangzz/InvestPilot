## 任务 09.1：报告中心

所属子项目：web

前置依赖：07（AI 报告已实现）

### 方案设计

**页面路由**：`/reports`

**API**：`GET /api/reports?runId=xxx`

- JOIN `recommendations` + `ai_reports`，返回列表
- 默认 `runId` = 最新 success run
- 可选 `runId` 切换历史 run
- 无推荐时返回空列表，前端显示空状态

**UI 布局**：

```
┌─────────────────────────────────────────┐
│  报告中心          [run 选择器 ▼]        │
├─────────────────────────────────────────┤
│  🔍 搜索股票代码/名称                    │
├─────────────────────────────────────────┤
│  📊 000001 平安银行  A 评分 78.5        │
│     趋势观点：股价处于强势上升通道…       │
│     [展开/收起]                         │
│  📊 601211 国泰海通  A 评分 91.6        │
│     …                                  │
└─────────────────────────────────────────┘
```

**功能点**：

- run 选择器：下拉列出所有 success 的 run，默认选最新
- 搜索框：实时过滤股票代码/名称
- 卡片列表：每条展开/收起 AI 报告全文
- 无 AI 报告时显示「AI 未生成报告」标识

**边界条件**：

| 场景 | 处理 |
|---|---|
| 无推荐数据 | 「暂无推荐报告」+ 引导运行筛选 |
| AI 未启用 | 「AI 未开启，报告为空」 |
| 某股票无 AI 报告 | 显示「未生成」标识 |
| run 切换 | 重新请求 API |

### 验收标准

- 报告列表正常展示，含股票信息和 AI 报告内容。
- run 选择器可切换历史 run。
- 搜索框可过滤。
- 无数据时有空状态引导。

## 任务 09.2：任务日志

所属子项目：web

前置依赖：05（API 任务层已完成）

### 方案设计

**页面路由**：`/tasks`

**API**：`GET /api/tasks`

- 查询 `system_tasks` 全部记录，按 `created_at DESC` 排序
- 全量返回，MVP 无分页

**UI 布局**：

```
┌─────────────────────────────────────────┐
│  任务日志    [全部] [成功] [失败] [运行中] │
├─────────────────────────────────────────┤
│  ✅ 2026-07-03 22:34  run-screening 0.5s │
│  ✅ 2026-07-03 22:33  calc_factors  0.3s │
│  ❌ 2026-07-03 22:30  update-data  失败   │
│     └─ error: pipeline crashed          │
└─────────────────────────────────────────┘
```

**功能点**：

- 状态筛选按钮：全部 / 成功 / 失败 / 运行中
- 失败任务展开查看 error_message
- 任务名称+时间+耗时显示

**边界条件**：

| 场景 | 处理 |
|---|---|
| 无任务记录 | 「暂无任务记录」 |
| 运行中任务 | 黄色标识，可轮询刷新 |
| 失败详情 | 点击展开/收起 |

### 验收标准

- 任务历史列表正常展示。
- 状态筛选按钮可用。
- 失败任务可查看错误详情。
- 无数据时有空状态提示。
