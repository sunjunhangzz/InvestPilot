# 12-第5阶段复盘闭环

> 2026-07 经 5 轮多角度方案评估 + 金融从业者视角补充后确定。

## 方案概要

复盘的目的是验证推荐是否真的有效。数据全部来自已有表，无需新数据采集。

## 复盘指标体系

### 收益指标

| 指标 | 计算 | 说明 |
|---|---|---|
| 推荐后 N 日收益 | (close[N天后] - close[推荐日]) / close[推荐日] | N = 1/3/5/10/15/20 |
| 平均收益 | 所有推荐股票的平均 | 分 A/B 类 |
| 胜率 | 正收益股票数 / 总数 | 分 A/B 类 |
| 盈亏比 | avg_gain / abs(avg_loss) | >1 表示盈利覆盖亏损 |
| Sharpe Ratio | (日均收益 - 0.015/252) / 收益标准差 × √252 | 年化，无风险利率假定 1.5% |
| 最大单日收益/亏损 | 最佳/最差单只单日 | 极端值识别 |

### 风险指标

| 指标 | 计算 | 说明 |
|---|---|---|
| 最大回撤 | 峰值到谷底的最大跌幅 | 已有（watchlist） |
| 连续亏损次数 | 连续出现负收益的股票数 | 判断策略稳定性 |
| 收益波动率 | 日收益标准差 | 稳定性度量 |

### 收益衰减曲线

折线图：横轴 1/3/5/10/15/20 日，纵轴累计平均收益。判断最优持有期。

### 市场环境适应性

| 环境 | 判断标准 | 分析 |
|---|---|---|
| 牛市 | 推荐日 MA60 上行 + close > MA60 | 分离统计收益 |
| 熊市 | MA60 下行 + close < MA60 | 同上 |
| 震荡 | 其他 | 同上 |

### 行业/因子归因

| 指标 | 计算 | 说明 |
|---|---|---|
| 行业表现排名 | 按行业分组求平均收益 | 发现优势行业 |
| 行业集中度 | 最大行业占比 | >40% 告警 |
| 因子 IC | 因子值与未来收益的相关系数 | 判断因子有效性 |

### 换手率

| 指标 | 计算 | 说明 |
|---|---|---|
| 重叠度 | 本次推荐 ∩ 上次推荐 / 总推荐 | 100% = 完全不变 |
| 新进数量 | 本次新增股票数 | |
| 退出数量 | 上次有、本次无的股票数 | |

## 任务拆分

### 任务 12.1：复盘 API

所属子项目：web

前置依赖：01-04 全部完成

**开发内容**：

- 创建 `app/web/app/api/backtest/route.ts`
- 支持 `?runId=xxx&days=20`
- 默认 `runId` = 最新 success run
- 返回所有统计指标 + chart 数据 + 行业/因子分解

**验收标准**：

- 返回收益/风险/胜率/盈亏比/Sharpe 等指标
- 返回 chart 数据（1-20 日累计收益数组）
- 收益衰减曲线数据正确
