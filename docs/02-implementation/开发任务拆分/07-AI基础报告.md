# 07-AI基础报告

## 任务 07.1：AI Provider 配置

所属子项目：shared / worker

前置依赖：06.3

开发内容：

- 支持 AI 开关。
- 支持 provider。
- 支持 model。
- 支持 API Key。

验收标准：

- 没有 API Key 时不报错。
- AI 关闭时跳过报告生成。

风险点：

- API Key 泄露。

## 任务 07.2：生成基础报告

所属子项目：worker

前置依赖：07.1

开发内容：

- 创建 `generate_report.py`。
- 只对最终推荐股票生成报告。
- 报告写入 `ai_reports`。

验收标准：

- 有 API Key 时能生成报告。
- AI 失败不影响推荐结果。
- 报告内容能在股票详情页展示。

风险点：

- token 成本失控。
- 模型返回格式不稳定。

## 任务 07.2a：限制 AI 输入和 token 成本

所属子项目：worker

前置依赖：07.2

开发内容：

- 只对最终推荐股票生成报告。
- 输入给模型的数据必须是结构化摘要。
- 不把全量历史行情直接传给模型。
- 限制单只股票输入长度。
- 记录模型名称和调用状态。

验收标准：

- 输入内容可控。
- 单次生成失败不会影响其他股票。
- 可以估算每日 token 使用量。

风险点：

- 把原始行情或长文本直接塞给模型导致成本失控。

## 任务 07.3：AI报告展示

所属子项目：web

前置依赖：07.2

开发内容：

- 股票详情展示 AI 解释。
- 推荐页展示 AI 状态。

验收标准：

- 已生成报告能展示。
- 未生成报告有提示。
