## 任务 06.4a：近 60 日价格走势图（2026-07 追加）

所属子项目：web / worker

前置依赖：06.4，02.2

> 06 阶段遗漏项，经三轮方案评审后确定方案。

### 方案设计

| 层 | 文件 | 说明 |
|---|---|---|
| 后端 API | `app/web/app/api/stocks/[code]/prices/route.ts` | GET 返回 `{ prices: [{ date, close, volume }] }` |
| 前端组件 | `app/web/components/PriceChart.tsx` | ECharts 收盘价折线图 + 成交量柱状图 |
| 详情页 | `app/web/app/stocks/[code]/page.tsx` | 在因子卡片下方引入 PriceChart |
| 依赖 | `npm install echarts` | |

**API 设计**：

```
GET /api/stocks/000001/prices?days=60
→ { ok: true, data: { prices: [{ date, close, volume }] } }
```

- 查询 `daily_prices` 表，WHERE code + trade_date ≤ latest，ORDER BY date，LIMIT days
- 走 (code, trade_date) 联合索引
- 数据不足时返回实际数量，不报错
- 股票不存在返回 404

**图表设计**：

- 上半部分：收盘价折线图（60 日）
- 下半部分：成交量柱状图
- 不做完整 OHLC 蜡烛图，MVP 以简洁为主
- 空数据显示「暂无行情数据」
- echarts 加载失败显示占位

### 验收标准

- 价格走势图和成交量图正常显示。
- 数据不足 60 日时显示实际天数。
- 无行情数据时显示提示。
- 观察池页面（第 5 阶段）可复用此 API。

### 边界条件

- 无行情 → `{ prices: [] }` → 前端「暂无行情数据」
- 不足 60 日 → 返回实际条数
- 请求不存在的股票 → 404
- echarts 加载失败 → 「图表加载失败」
