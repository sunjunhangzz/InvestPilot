# 06-前端页面

## 任务 06.1：创建页面框架

所属子项目：web

前置依赖：00.4

开发内容：

- 全局布局
- 导航
- 首页
- 今日推荐
- 股票详情
- 观察池
- 设置

验收标准：

- 页面可访问。
- 导航可用。

风险点：

- 页面先做太复杂导致拖慢主线。

## 任务 06.1a：使用 mock 数据搭页面

所属子项目：web

前置依赖：06.1

开发内容：

- 为首页、推荐页、股票详情、观察池准备 mock 数据。
- 先完成布局和状态展示。

验收标准：

- 不依赖真实 API 也能看到页面效果。
- mock 数据结构和 API 约定一致。

风险点：

- mock 字段和真实 API 不一致。

## 任务 06.1b：补齐 loading / empty / error 状态

所属子项目：web

前置依赖：06.1a

开发内容：

- 所有页面增加加载状态。
- 所有页面增加空数据状态。
- 所有页面增加错误状态。

验收标准：

- 断开 API 时页面有错误提示。
- 无推荐结果时页面有引导。

风险点：

- 只处理成功状态，真实使用时体验差。

## 任务 06.2：首页仪表盘
