# web 页面设计

## 首页仪表盘

展示：

- 数据更新时间
- 最新成功 run_id
- 最新推荐数量
- A类数量
- B类数量
- 观察池数量
- 最近任务状态

主要操作：

- 更新数据
- 运行筛选
- 生成 AI 报告

## 今日推荐

页面名称保持“今日推荐”，但数据来源是最近一次成功 `run_id`。

如果当天没有成功运行结果，页面应明确提示当前展示的是最近可用交易日结果。

表格字段：

- 排名
- 股票代码
- 股票名称
- 行业
- 最新收盘价
- 20日涨幅
- 60日涨幅
- 成交额
- 总评分
- 评级
- 风险标签

筛选：

- A类 / B类
- 行业
- 风险标签

## 股票详情

展示：

- 基础信息
- 近 60 日价格走势
- 成交额走势
- 因子评分
- 推荐理由
- 风险提示
- AI解释报告

## 观察池

展示：

- 股票代码
- 股票名称
- 首次推荐日期
- 已跟踪天数
- 当前状态
- 推荐后收益
- 最大回撤
- 退出原因

操作：

- 手动降级
- 手动退出观察
- 查看详情

## 系统设置

第一版设置：

- 推荐股票数量
- 是否启用 AI
- AI Provider
- API Key
- 数据库路径展示
- 报告目录展示
