# worker 数据采集

## 数据源

第一版优先使用 AkShare。

采集内容：

- A股股票列表
- 股票基础信息
- 日线行情

## 股票池规则

第一版只保留 A股主板股票。

默认排除：

- ST 股票
- 退市风险股票
- 停牌股票
- 上市不足 180 天股票
- 低流动性股票

## 行情数据

第一版使用日线数据。

原则：

- 使用前复权数据计算均线和收益率。
- 推荐基于最近可用交易日。
- 非交易日使用最近一个交易日数据。
- 数据缺失股票跳过并记录日志。

## 缓存

原始数据可以保存到：

```text
data/raw/
```

中间缓存可以保存到：

```text
data/cache/
```

缓存不能作为唯一数据源，数据库才是系统读取的主数据源。
