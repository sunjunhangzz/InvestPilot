# worker 观察池规则

## 目标

观察池用于保证推荐连续性。

系统不应该今天推荐、明天无理由消失。

## 进入规则

每日推荐股票默认进入观察池。

如果股票已经在观察池：

- 更新 `last_recommended_date`
- 更新 `latest_price`
- 更新 `tracking_days`
- 保留首次推荐日期

## 最小跟踪周期

每只推荐股票至少跟踪 5 个交易日。

未触发风险退出条件时，不因单日评分下降立即移出。

## 状态

```text
active      正在观察
hold        持续观察
downgraded  降级观察
exit        退出观察
blocked     风险阻断
```

## 退出条件

- 跌破 MA20 或 MA60。
- 短期涨幅过热且风险收益比下降。
- 流动性明显恶化。
- 出现明确风险事件。
- 达到最大观察周期但趋势未继续改善。

## 退出要求

退出观察必须记录 `exit_reason`。

不允许无原因移除观察池股票。
