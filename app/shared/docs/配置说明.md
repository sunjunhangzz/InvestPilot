# shared 配置说明

## config.json

`config.json` 保存第一版系统运行参数。

主要字段：

```text
databasePath
reportsPath
recommendationLimit
minRecommendationLimit
minTrackingDays
maxTrackingDays
factorWeights
filters
ai
```

## 路径配置

路径配置供 TypeScript 和 Python 共同读取。

不要在代码里硬编码数据库路径。

## 因子权重

第一版因子权重：

```json
{
  "trend": 0.35,
  "momentum": 0.25,
  "liquidity": 0.2,
  "risk": 0.2
}
```

后续调整权重需要记录版本。

## 筛选阈值

第一版筛选阈值：

```json
{
  "minListingDays": 180,
  "minAverageAmount20d": 100000000,
  "minPrice": 3,
  "maxPrice": 300
}
```

## AI配置

第一版默认 AI 关闭。

没有 API Key 时，系统仍能生成基础推荐。
