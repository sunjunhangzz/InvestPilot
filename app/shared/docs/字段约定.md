# shared 字段约定

## 状态字段

运行状态：

```text
pending
running
success
failed
cancelled
```

运行类型：

```text
manual
morning
afternoon
scheduled
```

观察池状态：

```text
active
hold
downgraded
exit
blocked
```

推荐评级：

```text
A
B
C
```

报告类型：

```text
basic
stock
daily
```

## 命名原则

- 数据库字段使用 snake_case。
- TypeScript 类型可以使用 camelCase。
- API 返回给前端时使用 camelCase。
- Python 内部优先使用 snake_case。

字段转换必须集中处理，避免到处手写。
